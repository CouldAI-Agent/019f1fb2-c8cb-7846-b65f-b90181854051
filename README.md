# Celestial Noir Vedic Dashboard

A comprehensive, multi-dimensional Vedic Kundali analysis app built in Flutter. This application serves as a "digital sanctuary of destiny," rendering an intricate birth chart and providing deep astrological insights based on the user's exact birth details.

## Features

- **Celestial Noir Theme:** A visually stunning aesthetic featuring deep purples, golds, and midnight blues with shimmering star animations.
- **North Indian Lagna Chart:** A custom-painted, SVG-style SVG grid visualizing the planetary positions.
- **Interactive Navigation:** Dedicated tabs for Overview, Career & Karma, Love & Union, Health & Vitality, Wealth/Artha, and Remedies.
- **Exhaustive Vedic Analysis:** In-depth astrological breakdown covering Lagna, Nakshatras, Mahadasha, wealth alignment, and personalized Upayas.

## Tech Stack

- **Framework:** Flutter (Mobile, Desktop, and Web)
- **UI & Visualization:** Core Flutter `CustomPaint` for the intricate North Indian style chart, along with `AnimationController` for dynamic starfields.

## Setup Instructions

1. Ensure you have [Flutter installed](https://flutter.dev/docs/get-started/install).
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred platform.
